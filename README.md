# Diary
## Introduction
A secret diary application me using MERN stack, that lets you save encrypted notes.

## Getting Started
- Clone the repo
- Run `npm i && npm i --prefix frontend` (Installation of dependencies)
- Create `.env` file using the template given in `.envexample`
